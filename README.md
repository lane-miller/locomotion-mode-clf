# locomotion-mode-clf

Comparing traditional DSP vs ML methods for classifying locomotion modes from inertial measurement unit (IMU) sensor data


## Overview

This project uses the Camargo et al. (2021) dataset (open-source lower-limb biomechanics and wearable sensor data from 22 able-bodied adults across multiple locomotion modes and terrains). Only data from 15 subjects used here.

Analysis stages:

2. **ML approach:** 

The goal is a proof-of-concept demonstrating the data science process: signal exploration, feature engineering, model development, and evaluation / comparison.

## Locomotion Modes

- Level-ground walking (multiple speeds)
- Treadmill walking (multiple speeds)
- Stair ascent / descent (4 step heights)
- Ramp ascent / descent (6 inclinations)
